📊 Dashboard

Today's snapshot: sales, costs, margin, anomalies flagged, low stock alerts

📦 Inventory

Current stock levels, add via receipt photo, manual adjustments, supplier links

🍽️ Menu

Menu items with recipes, cost per item, margin %, edit ingredient quantities

💵 Sales

Import from POS (photo or API), daily/weekly view, profit per item breakdown

🔍 Insights

Variance analysis, anomaly flags needing explanation, trends over time, market suggestions (stub)

🧪 Model

What-if scenarios: new menu item, supplier change, price adjustment — see projected impact

Mobile: Bottom tab navigation (5 tabs max, combine Dashboard + Insights). Quick-action FAB for "Scan Receipt".
