Variance Detection & Anomalies

How the system spots problems and learns over time

The Calculation
Expected usage = Σ (sales × recipe quantities)
Actual usage = Starting inventory − Ending inventory + Purchases
Variance = Actual − Expected

Run daily or weekly. Positive variance = used more than expected. Negative = used less (rare, possibly data entry error).

  • Track variance patterns by day-of-week, ingredient, menu item
